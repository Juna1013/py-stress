# test_01.py - 車体骨格部材の曲げ応力シミュレーション

## 概要

Bピラーを想定した車体骨格部材（角パイプ）の3点曲げにおける応力シミュレーションを行い、軟鋼とホットスタンプ材の比較を行うプログラムです。

## 依存ライブラリ

- `numpy` - 数値計算
- `matplotlib` - グラフ描画

## 計算モデル

### パラメータ設定

| パラメータ | 値 | 説明 |
|------------|------|------|
| L | 1000 mm | 支点間距離 |
| W | 100 mm | 断面の幅 |
| H | 150 mm | 断面の高さ |
| σy (軟鋼) | 440 MPa | 軟鋼の降伏点 |
| σy (ホットスタンプ) | 1500 MPa | ホットスタンプ材の降伏点 |
| t (軟鋼) | 2.0 mm | 軟鋼の板厚 |
| t (ホットスタンプ) | 1.2 mm | ホットスタンプ材の板厚 |
| 荷重範囲 | 0〜50 kN | 負荷荷重 |

### 計算式

#### 断面二次モーメント（中空長方形断面）

$$I = \frac{WH^3 - (W-2t)(H-2t)^3}{12}$$

#### 最大曲げモーメント（中央集中荷重）

$$M = \frac{F \cdot L}{4}$$

#### 最大曲げ応力

$$\sigma = \frac{M \cdot y}{I}$$

ここで $y = H/2$（中立軸からの距離）

## 出力

1. **グラフ**: 荷重に対する発生応力の比較グラフ（PNG形式）
2. **コンソール出力**: 軽量化の試算結果
   - 各材料の断面積
   - 重量削減率

## 使用方法

```bash
python src/test_01.py
```

## 目的

ホットスタンプ材（高強度材）を使用することで、板厚を薄くしても同等以上の強度を維持でき、軽量化が可能であることを示す。
